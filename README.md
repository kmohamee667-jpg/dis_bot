# GALAXY ROLE SHOP 💎✨

نظام متجر الرتب والعملات المتكامل لسيرفر GALAXY.

## 🚀 كيفية التشغيل / How to Start
1. ضع التوكن في `.env`:
   ```
   TOKEN=your_bot_token_here
   ```
2. ثبت التبعيات:
   ```
   npm install
   ```
3. نشر الأوامر للسيرفر الجديد:
   ```
   node deploy-commands.js
   ```
4. شغل البوت:
   ```
   npm start
   ```

## 📖 الأوامر المتاحة / Available Commands

### 💰 نظام العملات (Economy)
- `/coins`: عرض بطاقة رصيدك الحالية (أو رصيد عضو آخر للمسؤولين).
- `/daily`: الحصول على المكافأة اليومية (1000 كوين).
- `/give`: إرسال عملات لعضو آخر (أو لنفسك إذا كنت مسؤولاً).

### 🏪 متجر الرتب (Role Shop)
- `/shop`: عرض المتجر وشراء الرتب (لـ dev فقط لتثبيت المتجر).
- `/add-role`: إضافة رتبة جديدة للمتجر (للمسؤولين فقط).
- `/delete-role`: حذف رتبة من المتجر (للمسؤولين فقط).

### 👮 الإدارة (Admin)
- `/rm-coins`: تصفير الكوينات لمستخدم معين أو للجميع (للمسؤولين فقط).
